# 城市乐园B端门店后台管理系统 - 使用指南

欢迎使用城市乐园B端门店后台管理系统原型！本指南将帮助您快速了解系统的主要功能和使用方法。

## 系统概览

本系统是一个移动端门店后台管理应用，专为城市乐园的直营门店和加盟商设计，提供全方位的门店运营管理功能。

## 主要功能模块

### 1. 首页 🏠

首页是您的运营指挥中心，提供：

- **门店切换**：顶部可快速切换当前管理的门店
- **消息通知**：实时查看重要的系统通知和运营提醒
- **主要功能入口**：快速访问订单管理、门票套餐、营销工具、终端设备
- **常用功能**：会员管理、商品管理、游乐项目等10个常用功能入口
- **实时数据看板**：
  - 今日营业数据（营业额、订单数、游玩人数）
  - 会员概况（总会员、新增、活跃、VIP）
  - 游玩分析（游玩人次、核销率监控）

### 2. 分析 📊

数据分析中心，帮助您深入了解门店运营情况：

- **营收分析**：查看营收趋势和构成（开发中）
- **会员分析**：了解会员增长和活跃情况（开发中）
- **游玩分析**：分析各项目游玩情况，监控逃票（开发中）
- **库存分析**：掌握商品库存状态（开发中）

### 3. 商城 🛒

供应链管理平台：

- **采购商城**
  - 左侧商品分类导航
  - 游乐设备、运营物资、IP周边等商品采购
  - 购物车和采购订单管理（开发中）

- **租赁平台**
  - 设备租赁服务
  - 灵活的租赁方案（开发中）

### 4. 我的 👤

个人中心和系统设置：

- **账号信息**：头像、姓名、手机号
- **等级系统**：金牌加盟商等级展示
- **门店管理**：管理多家门店信息
- **资产信息**：账户余额、积分查看
- **融资推荐**：设备采购分期、经营贷款等融资产品
- **系统设置**：账户设置、系统配置、常见问题、客服咨询

## 核心功能详解

### 订单管理

**路径**：首页 → 订单管理

- **销售订单**：查看所有销售订单，支持按状态筛选
- **核销订单**：管理门票核销记录
- 订单详情包含：订单号、客户信息、金额、状态、时间

### 门票套餐

**路径**：首页 → 门票套餐

- **门票管理**
  - 查看所有门票类型（儿童票、成人票、亲子票等）
  - 显示价格、库存、销量
  - 支持添加和编辑门票

- **套餐管理**
  - 创建组合套餐（欢乐亲子套餐、生日派对套餐等）
  - 设置套餐价格和包含内容
  - 跟踪套餐销售情况

### 营销工具

**路径**：首页 → 营销工具

集成多种营销手段：

- **分销管理**：员工分销、用户分销、渠道分销
- **券管理**：优惠券创建、派发、核销
- **促活专区**：限时优惠、拼团活动、砍价活动、领券中心
- **互动专区**：幸运抽奖、小程序广告设置
- **拉新专区**：新人专享、拉新有礼

### 终端设备

**路径**：首页 → 终端设备

- **设备状态概览**：运营中、维护中、已停用设备统计
- **游乐项目管理**：
  - 查看所有游乐项目状态
  - 显示容量和等待时间
  - 快速进行设备维修和终端配置

### 会员管理

**路径**：首页 → 会员管理

- **会员数据概览**：总会员、新增、活跃、VIP统计
- **会员列表**：
  - 搜索会员（姓名、手机号）
  - 查看会员信息、余额、积分
  - VIP会员标识

### 商品管理

**路径**：首页 → 商品管理

- **商品列表**：B.Duck玩偶、纪念品等
- **库存预警**：低库存商品提醒
- **商品信息**：价格、库存、销量、分类

### 游乐项目

**路径**：首页 → 游乐项目

- 项目统计：总数、运营中、维护中
- 项目详情：容量、状态、等待时间
- 项目管理：添加、编辑、状态变更

### 分销管理

**路径**：首页 → 营销工具 → 分销管理

三种分销模式：
- **员工分销**：激励员工推广
- **用户分销**：会员推荐奖励
- **渠道分销**：合作渠道管理

每个模式显示：分销人数、销售额、佣金统计

### 券管理

**路径**：首页 → 营销工具 → 券管理

- **券种管理**：创建满减券、折扣券、免费券
- **送券活动**：设置自动送券规则
- **派送记录**：查看券的发放情况
- **核销记录**：跟踪券的使用情况

### 促活专区

**路径**：首页 → 促活专区

四大活动类型：
- **限时优惠**：设置时间限制的优惠活动
- **拼团活动**：多人组团享优惠
- **砍价活动**：邀请好友帮忙砍价
- **领券中心**：集中展示可领取的优惠券

### 自营商城

**路径**：首页 → 自营商城

- **商城概览**：订单数、销售额、待发货统计
- **订单管理**：处理线上商城订单
- **发货管理**：物流信息录入和跟踪
- **商城装饰**：轮播图、分类、推荐商品设置

### 我要融资

**路径**：首页 → 我要融资

提供多种融资方案：
- **设备采购分期**：0首付，月供低至3000元
- **经营贷款**：最高100万额度
- **供应链金融**：灵活周转，随借随还

### 乐园学院

**路径**：首页 → 乐园学院

- **加盟商等级**：显示当前等级和学习进度
- **学习资源**：运营课程、视频教程
- **运营咨询**：
  - 在线客服
  - 预约运营顾问
  - 加盟商社群
  - 运营资料下载

## 使用技巧

### 1. 快速切换门店

如果您管理多家门店，可以：
1. 点击首页顶部的门店名称
2. 从下拉列表中选择目标门店
3. 所有数据将自动切换到选中门店

### 2. 查看详细分析

从首页数据看板快速跳转到详细分析页：
1. 滚动到首页底部
2. 点击"查看详细分析报告"
3. 在分析页面切换不同维度的数据

### 3. 快速核销订单

1. 首页 → 订单管理
2. 切换到"核销订单"标签
3. 查看已核销的订单记录

### 4. 设置营销活动

1. 首页 → 营销工具
2. 选择合适的营销手段（券、拼团、砍价等）
3. 创建并发布活动
4. 在营销数据概览中查看效果
