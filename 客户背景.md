# 公司概况
城市乐园（CityFuns）是中国领先的城市中小型主题游乐解决方案提供商，专注于为商业综合体、市政公园、文旅景区等场景打造融合亲子娱乐与城市轻文旅的创新型业态。公司以“Make a Playful World”为愿景。

# 核心模式与优势
公司通过IP驱动+全产业链服务，提供轻资产、高流量、强变现的解决方案。其核心优势在于提供从IP孵化、设备研发、系统开发到落地运营的一站式服务，并拥有如B.Duck小黄鸭等全球化IP的独家授权，具备跨界联名与场景定制能力。

# 核心产品体系：IPESO模型
公司以IPESO模型为核心，整合五大业务模块，打造可复制的主题娱乐生态：
•   IP主题运营：独家运营B.Duck等国际IP，提供IP主题乐园、城市造节活动（如趴趴鸭派对）、跨界联名营销等服务。   
•   产品设计：提供模块化产品矩阵，涵盖无动力/动力设备、艺术装置及智慧游乐系统，适配500-5000㎡场景。
    
# 业务模式
业务模式涵盖四大核心板块：
1.  经营授权合作：提供整店输出和轻资产联营两种模式。
2.  IP开发与联名：通过主题设备授权、智慧零售产品开发及跨业态品牌联名提升变现能力。 
3.  运营托管服务：提供全周期运营支持。
4.  全球化布局：国内加速落地，并计划于2025年拓展至东南亚及美国市场。

# 愿景
最终目标是以前沿的IP和全产业链创新模式，赋能城市空间，为亲子家庭打造触手可及的欢乐体验。

# 主要产品
-室内儿童乐园、家庭娱乐中心
-室外公园级游乐园：引入中小型游乐设备（如旋转木马、轨道小火车），融入IP主题包装，打造低刺激度、高趣味性的亲子互动场景
-室外商业乐园：针对城市商业广场空间特性，打造 “小而精、快引流、高坪效” 的室内外动力游乐集群。通过模块化动力设备组合，快速激活广场闲置空间（屋顶、中庭、下沉广场等）
